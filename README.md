# Singapore Weather Outlook Dashboard

A premium, real-time 4-day weather forecast dashboard for Singapore, powered by the [data.gov.sg API](https://api-open.data.gov.sg/v2/real-time/api/four-day-outlook).

## ✨ Features
- **Real-time Data**: Fetches the latest 4-day forecast outlook.
- **Premium Design**: Dark-themed, glassmorphic UI with animated background elements.
- **Visual Indicators**: Clear weather icons, temperature ranges, humidity, and wind details.
- **Robust Fallback**: Includes a mock data system to maintain a beautiful UI even when the API is rate-limited.
- **Fully Responsive**: Optimized for desktop and mobile devices.

## 🚀 How to use
Simply clone the repository and open `index.html` in any modern web browser.

```bash
git clone https://github.com/joiboi/sg-weather-outlook.git
cd sg-weather-outlook
open index.html
```
